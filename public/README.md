# Images for 4 Pics 1 Word Game

Place your Egypt-related images in this `public` folder.

## Image Naming Convention

Based on the data in `src/data/fourPicsData.json`, you should name your images according to the answer word. For example:

- For "PYRAMID" → `pyramid1.png`, `pyramid2.png`, `pyramid3.png`, `pyramid4.png`
- For "NILE" → `nile1.png`, `nile2.png`, `nile3.png`, `nile4.png`
- For "CAIRO" → `cairo1.png`, `cairo2.png`, `cairo3.png`, `cairo4.png`
- etc.

## Current Image References

The game currently references placeholder images. Update `src/data/fourPicsData.json` to point to your actual images.

For example, change:
```json
"images": ["/placeholder1.png", "/placeholder2.png", "/placeholder3.png", "/placeholder4.png"]
```

To:
```json
"images": ["/pyramid1.png", "/pyramid2.png", "/pyramid3.png", "/pyramid4.png"]
```

## Image Requirements

- Format: PNG, JPG, or JPEG
- Recommended size: 400x400px or larger (square format works best)
- Content: Each set of 4 images should relate to the answer word

## Where to Find Egypt-Related Images

You can find free images from:
- Unsplash (unsplash.com) - Search for "Egypt", "Pyramid", "Nile", etc.
- Pexels (pexels.com) - Free stock photos
- Pixabay (pixabay.com) - Free images

Make sure to use images that are free to use or properly licensed.
